# AWS Serverless Student Registry Application 🚀

A full-stack serverless web application deployed on AWS using Amplify, API Gateway, Lambda, and DynamoDB.

## Live Demo

🌐 Live Application: `https://master.d15qybixo5lzep.amplifyapp.com`

## AWS Services Used

* AWS Amplify (Frontend Hosting)
* Amazon API Gateway (REST API)
* AWS Lambda (Backend Compute)
* Amazon DynamoDB (NoSQL Database)
* GitHub (Source Control & Deployment)

## Architecture

```text
User
  ↓
AWS Amplify
  ↓
API Gateway
  ↓
AWS Lambda
  ↓
Amazon DynamoDB
```

## Features

* Add Student Records
* Retrieve Student Records
* Serverless Architecture
* Fully Managed AWS Services
* Automatic Deployment via AWS Amplify
